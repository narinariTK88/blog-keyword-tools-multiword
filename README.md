# 🚀 ブログキーワード・サジェスト調査ツール

Googleのサジェスト（あ〜ん、a〜z、0-9）を全自動で取得し、ブログ記事のネタになる「お宝キーワード」を抽出するツールです。

## 🛠 機能の概要

1.  **キーワード自動調査**: `keywords.txt` にワードを書いて保存するだけで、約80パターンのサジェストを自動取得。
2.  **お宝キーワード（TREASURE）抽出**: 「おすすめ」「比較」「やり方」などのインテント（意図）が含まれるワードだけを自動選別。
3.  **全件まとめ機能**: 溜まった複数のCSVファイルをボタン一つで1つのファイルに統合（Ubersuggestへの一括投入用）。

---

## 📂 フォルダ構成

-   `keywords.txt`: 調査したいキーワードを入れる場所（1行1ワード）。
-   `data/`: キーワードごとの調査結果が保存されます。
    -   `KW_...`: 全サジェスト結果。
    -   `TREASURE_...`: インテント修飾語が含まれる厳選結果。
-   `summary/`: 全データを1つにまとめたCSVが保存されます。

---

## 🚀 使い方

### 1. 調査を開始する
1.  `keywords.txt` を開き、調べたいキーワードを入力して保存（Commit）します。
2.  GitHub Actionsが自動で起動し、数分後に `data/` フォルダへ結果が保存されます。

### 2. まとめて分析する（Ubersuggest連携）
1.  GitHubの **[Actions]** タブをクリック。
2.  左メニューの **[Combine All CSVs]** を選択。
3.  右側の **[Run workflow]** ボタンを押すと、`summary/` フォルダに全データを統合したファイルが生成されます。
4.  そのファイルを Ubersuggest の「キーワード一括分析」に貼り付けて調査してください。

---

## 💡 インテント抽出（お宝ワード）の対象
現在、以下のワードが含まれるものを自動で「お宝」として抽出しています。
> おすすめ、比較、やり方、始め方、初心者、レビュー、とは、理由、仕組み、作り方、最安値、ランキング、選び方、違い

※ `main.py` の `treasure_words` リストを編集することで、自由に追加・変更が可能です。

---

## ⚠️ 注意事項
- 本ツールは個人利用を目的としています。短時間に大量のキーワードを詰め込みすぎると、Googleから一時的に制限を受ける可能性があります（通常、数個〜十数個程度なら問題ありません）。
